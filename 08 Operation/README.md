# 08 Operation

# 1. 산술 연산자

## 더하기

`+`

- `+`는 연결 연산자로도 사용된다.

## 빼기

`-`

## 곱하기

`*`

## 나누기

`/`

연산의 몫을 반환한다.

- 정수 / 정수 = 정수
- 정수 / 실수 = 실수

## 나머지

`%`

# 2. 대입 연산자

## 대입

`=`

## 증가

`+=`

A += B == A = A + B;

## 감소

`-=`

A -= B == A = A - B;

## 곱

`*=`

A *= B == A = A * B;

## 나누기

`/=`

A /= B == A = A / B;

# 3. 비교 연산자

**논리형을 반환**한다.

## 등호

`==`

## 부등호

`>` , `>=`, `<`, `<=`

## 부등

`!=`

# 4. 논리 연산자

## 그리고

`&&`

## 혹은

`||`

## 아니다

`!`
